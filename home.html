Github link -> movie App

Summary of the project

The movie app project built in React using TMDB API is a web application that allows users to search for movies, view their details and ratings, and create their own watchlists. The app utilizes the TMDB API to fetch data about movies, such as their titles, release dates, overviews, posters, and ratings.

The front-end of the app is built using React, a popular JavaScript library for building user interfaces, and styled with CSS.

Users can search for movies by typing in keywords or selecting specific genres, and the app dynamically displays a list of results that match their search criteria. Users can then click on a movie to view its details, such as its plot summary, release date, and user ratings.

Github link -> Text-to-speech

Summary of the project

The project is a web application built using JavaScript, Bootstrap, HTML, and CSS, and it utilizes the SpeechSynthesis API. This API allows the user to retrieve information about the available synthesis voices on the device, start and pause speech, and other commands related to text-to-speech conversion. The application can be used to enter text and then convert it to speech using the available synthesis voices on the device. The user can control the playback of the speech with commands such as pause, resume, and stop. The application has a user-friendly interface and can be used to convert text to speech for various purposes, such as accessibility or language learning.
